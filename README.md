# 高齢者とご家族の安心相談窓口

三鷹市周辺で、見守り・空き家・住まい・相続の入口相談を確認できる静的サイトです。名刺、手紙、紹介を受け取った方や、遠方のご家族が活動実態と連絡先を確認できる受け皿として作成しています。

## 公開方法

Cloudflare Pages で新規プロジェクトとして公開します。

- GitHub repository: `t-iguchi0802/mitaka-anshin-soudan`
- Project name: `mitaka-anshin-soudan`
- Framework preset: `None`
- Build command: 空欄
- Build output directory: `/`
- Production branch: `main`

既存の `sunarrows-consulting-site`、`sunarrows-tky.co.jp`、Netlify、STUDIO は使いません。

## 電話番号・フォームURLの変更

- 電話番号は `index.html` 内の `03-4500-5071` と `tel:0345005071` を変更します。
- 受付時間は `index.html` 内の `9:00〜18:00` を変更します。
- フォームURLは `index.html` のお問い合わせセクションにある `href="#"` を GoogleフォームURLへ差し替えます。
- 料金表や「三鷹市内は交通費をいただいていません」の補足を変更する場合は、`index.html` の料金セクションを編集します。

## 活動記録の更新

`activities.json` に活動記録を追加します。トップページには日付降順で最新3件が表示されます。

```json
{
  "date": "2026-07-02",
  "category": "地域活動",
  "title": "活動タイトル",
  "summary": "短い説明文",
  "image": "assets/activities/example.jpg",
  "alt": "写真の説明",
  "url": "activities/2026-07-example.html"
}
```

## 活動写真の追加

活動写真は `assets/activities/` に保存し、`activities.json` の `image` にパスを記載します。画像がない場合や読み込めない場合は、カード内に落ち着いた仮表示が出ます。

個人が特定される高齢者の顔写真は原則掲載しません。掲載する場合は、必ず本人または関係者の許可を得てください。基本は、お弁当、手元の作業、風景、配布準備、建物外観の一部など、個人が特定されない写真を使います。

## 活動記事の追加

個別記事は `activities/` にHTMLファイルとして追加します。既存の `activities/2026-07-activity-001.html` などをコピーし、タイトル、日付、本文を変更してください。

## 専門領域の注意

このサイトは入口相談の案内です。法律・税務・医療・介護などの専門判断は行わず、必要に応じて士業、医療・介護機関、行政窓口などの専門機関へつなぎます。パスワード、暗証番号、金融情報は預かりません。

訪問先で作業や役務提供の契約を行う場合は、特定商取引法の訪問販売に該当する可能性があります。作業内容、料金、実費、担当者、キャンセル条件、クーリング・オフ等の説明が必要になる場合があるため、事前に書面またはメッセージで内容を明確にしてください。

電気まわりの作業は、第二種電気工事士として対応できる範囲を確認したうえで案内してください。業として行う電気工事は登録等が関係する場合があるため、必要に応じて登録電気工事業者などの専門業者へつないでください。このサイトは、介護保険サービス、医療行為、身体介助を提供するものとして表示しないでください。
